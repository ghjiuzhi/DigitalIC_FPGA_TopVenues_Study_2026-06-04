# paper_01 - Memristive Tabu Learning Neuron Generated Multi-Wing Attractor With FPGA Implementation and Application in Encryption

## 一句话定位
这篇把忆阻器的分段非线性状态函数嵌入 tabu learning neuron，构造可调多翼混沌吸引子，并用 FPGA 实现与图像加密应用证明其电路/系统价值。

## 摘要写法
摘要先给出器件/模型优势：忆阻器天然适合构造非线性神经动力学。随后指出本文动作：把分段非线性忆阻状态函数引入 TLN 模型。中段用“可调翼数、复杂动力学、双/多翼混沌”概括理论贡献，最后落到 FPGA 与加密应用，形成“模型新颖性 -> 硬件可实现 -> 应用有效性”的闭环。

## Introduction 写法
引言不是从应用直接讲起，而是先回顾忆阻器、混沌系统和 tabu learning neuron 的研究线，再指出现有模型在吸引子形态调控和硬件验证上的不足。贡献列表承担很强的导航作用：提出模型、分析动力学、给出硬件实现、展示加密应用。

## 方法/架构写法
主体按“数学模型 -> 动力学分析 -> 离散化 -> FPGA 实现 -> 应用验证”推进。TCAS-I 常见的写法是先把系统方程和参数定义清楚，再用相图、分岔、Lyapunov 等图证明确实产生新动力学，最后再谈硬件映射，避免只停留在仿真模型。

## 实验/测量写法
实验强调两类证据：一类是非线性系统本身的动力学证据，另一类是 FPGA 资源/输出波形/加密性能证据。它的说服力来自“理论图像和硬件输出能互相对上”，而不是单纯列指标。

## 图表套路
图表承担故事线：模型框图说明新系统，吸引子相图展示现象，分岔图和 Lyapunov 图证明复杂性，FPGA 框图与波形图证明可实现，最后用加密效果图收束应用。

## 可模仿写法
可模仿它的“多层验证”结构：新模型论文不能只写推导，要让读者看到数学性质、仿真现象、硬件实现和应用场景逐层闭合。
