# Table Role Taxonomy

## 表的常见角色

- Setup table：平台、参数、工具链、样本。
- Parameter table：设计参数或实验变量。
- SOTA table：和相关工作比较。
- Result table：核心结果数字。
- Ablation/control table：排除替代解释。
- Traceability table：artifact、bitstream、route、日志。
- Limitation table：适用范围和未覆盖项。

## 主文表格保留标准

表格应满足至少一项：

- 支撑摘要或贡献列表中的强 claim。
- 审稿人必须看到它才能相信实验边界。
- 它比较了 prior work 或 baseline。
- 它排除了一个关键替代解释。

## RO-TRNG 当前稿件风险

表格较多时，读者可能不知道哪张是主证据。应显式标出 core evidence、supporting control、traceability/archive 三层。
